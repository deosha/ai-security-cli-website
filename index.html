Quick Start

Up and Running in Minutes

Install, scan, test. It's that simple.

1

Install

Install from PyPI with pip

pip install ai-security-cli
2

Scan Your Code

Static analysis for code vulnerabilities

# Scan a directory
ai-security-cli scan ./my-project -o html

# Filter by severity
ai-security-cli scan ./src --severity high
3

Test Live Models

Runtime testing against actual LLMs

# Test OpenAI model
ai-security-cli test -p openai -m gpt-4

# Test local Ollama model
ai-security-cli test -p ollama -m llama2
