# aisentry Website

Marketing website for [aisentry](https://github.com/deosha/aisentry) - a unified AI/LLM security scanner.

## Pages

- **Homepage** (`index.html`) - Problem statement, solution architecture, quick start
- **Documentation** (`docs.html`) - OWASP LLM Top 10, CLI reference, provider setup

## Deployment

Deployed via AWS Amplify. See `amplify.yml` for build configuration.
